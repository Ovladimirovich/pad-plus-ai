## Интеграция бота с проектом (на будущее)

Когда будет готов токен, бота можно подключить к проекту через Python:

```python
# backend/bot/telegram_bot.py
from telegram import Update, BotCommand
from telegram.ext import Application, CommandHandler, ContextTypes

BOT_TOKEN = "..."  # из .env

async def start(update: Update, context: ContextTypes.DEFAULT_TYPE):
    await update.message.reply_text("👋 Привет! Я бот PAD+ AI...")

async def github(update: Update, context: ContextTypes.DEFAULT_TYPE):
    await update.message.reply_text("🐙 github.com/Ovladimirovich/pad-plus-ai")

def main():
    app = Application.builder().token(BOT_TOKEN).build()
    app.add_handler(CommandHandler("start", start))
    app.add_handler(CommandHandler("github", github))
    app.run_polling()
```

**Варианты для продакшена:**

1. **Минимальный:** отдельный процесс `python -m backend.bot.telegram_bot` (не требует изменений в FastAPI)
2. **Встроенный:** запускать бота в фоновом таске при старте `main.py` (но осторожно с asyncio)
3. **Render Cron Job:** отдельный сервис на Render (бесплатный план — 1 процесс)

Для первого запуска достаточно варианта 1 — запустить локально или на VPS.
