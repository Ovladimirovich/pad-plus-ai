# PAD+ Impulse Research Package

Для получения полного пакета исследований обращайтесь к разработчикам PAD+.

## В состав пакета входит

### 1. Данные экспериментов (I-002 — I-005)

- **raw_responses.json** — все ответы LLM на 7 вопросов для каждого раунда
- **REPORT.md** — анализ гипотезы, методики, результатов и выводов
- **HYPOTHESIS.md** — предэкспериментальная формулировка гипотезы

### 2. Методология

- 7 стандартных вопросов для воспроизведения
- Протокол смены импульса
- Контролируемые переменные (модель, температура, окружение)

### 3. Инструментарий

- Полный код PAD+ AI пайплайна для запуска локально
- Скрипты для автоматизации экспериментальных сессий
- Инструкция по настройке окружения

### 4. Права

- Пакет предоставляется для академического и некоммерческого использования
- При публикации результатов ссылка на PAD+ Impulse Research Program обязательна

## Контакты

Через платформу https://pad-plus-ai.onrender.com
